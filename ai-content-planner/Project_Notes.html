AI Content Planner - Quick Reference Guide

1. What is it?
A production-ready SaaS web application for creators to generate, save, and manage their content ideas using Artificial Intelligence.

2. Tech Stack

  • Frontend: React 19, Vite, TailwindCSS (for styling), Lucide (for icons).
  • Backend / Platform: InsForge MCP (handles Authentication, Database, AI connections, and edge functions).
  • Language: TypeScript (Full-stack type safety).

3. Key Things to Remember

  • Authentication: Needs InsForge setup. Only logged-in users can generate and save ideas.
  • Environment Variables: The app relies on the .env file (using VITE_INSFORGE_BASE_URL and VITE_INSFORGE_ANON_KEY).
  • Starting the project: Just run npm run dev in your terminal.
